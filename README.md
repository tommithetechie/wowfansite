# Azeroth Echoes ⚔️  
A dusty little corner of the web where old raid nights, snowy zones, and late-night forum energy still glow.

## About This Project
This fansite started life years ago from a free World of Warcraft fansite kit, then sat in stasis for a long while before getting a recent polish pass.  
Most of the content is intentionally old-school (and definitely out of date), but the project lives on as a fun nostalgia archive and a learning playground for static-site tinkering.

## Previewing the Site
This is a static site—just open HTML files directly in your browser.

- Main modern page: `index.html`
- Legacy kit templates: `web/index.html`, `web/index_2column.html`, `web/index_horizontal.html`, etc.

Tip: double-click the file, or drag it into a browser window.

## Where to Edit Content
- Main homepage structure/content: `index.html`
- Main styling: `style.css`
- Legacy template pages: `web/*.html`
- Legacy template styles: `web/css/*.css`
- Images and visual assets:  
  - `screenshots/`
  - `wallpapers/`
  - `logos/`
  - `artwork/`
  - `avatars/`
  - `unit_renders/`
